# DF0009: Command Already Registered

## Message

> Command "`{id}`" is already registered

## Cause

`DevToolsCommandsHost.register()` rejects duplicate command ids.

## Fix

Use a unique id or call `unregister(id)` first.

## Source

- [`packages/devframe/src/node/host-commands.ts`](https://github.com/vitejs/devtools/blob/main/devframe/packages/devframe/src/node/host-commands.ts) — `DevToolsCommandsHost.register()` throws `DF0009` when a command id is already in use.
