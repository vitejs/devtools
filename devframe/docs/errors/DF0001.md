# DF0001: Dock Already Registered

## Message

> Dock with id "`{id}`" is already registered

## Cause

`DevToolsDockHost.register()` rejects duplicate ids by default.

## Fix

Pass `force: true` as the second argument to intentionally overwrite, or choose a unique dock id.

## Source

- [`packages/devframe/src/node/host-docks.ts`](https://github.com/vitejs/devtools/blob/main/devframe/packages/devframe/src/node/host-docks.ts) — `DevToolsDockHost.register()` rejects duplicate dock ids; throws `DF0001` when one already exists.
