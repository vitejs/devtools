# DF0003: Dock Not Registered

## Message

> Dock with id "`{id}`" is not registered. Use register() to add new docks.

## Cause

`DevToolsDockHost.update()` was called with an id that has no matching registration.

## Fix

Call `register()` first, or check the id against `dockHost.values()` before updating.

## Source

- [`packages/devframe/src/node/host-docks.ts`](https://github.com/vitejs/devtools/blob/main/devframe/packages/devframe/src/node/host-docks.ts) — `DevToolsDockHost.update()` throws `DF0003` when called with an id that has no matching registration.
