# DF0011: Command Not Registered

## Message

> Command "`{id}`" is not registered

## Cause

`DevToolsCommandsHost.update()` or `.unregister()` was called with an unknown id.

## Fix

Register the command first, or check `commandsHost.list()` before operating on it.

## Source

- [`packages/devframe/src/node/host-commands.ts`](https://github.com/vitejs/devtools/blob/main/devframe/packages/devframe/src/node/host-commands.ts) — `DevToolsCommandsHost.update()` and `unregister()` both throw `DF0011` when called with an unknown command id.
