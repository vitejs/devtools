# DTK0052: Dock Not Registered

## Message

> Dock with id "`{id}`" is not registered. Use register() to add new docks.

## Cause

`ctx.docks.update(view)` was called with an `id` that has not been registered. Updates require the dock to exist first.

## Fix

Call `ctx.docks.register(...)` before `update`, or guard with `ctx.docks.views.has(id)`:

```ts
if (ctx.docks.views.has('my-plugin'))
  ctx.docks.update({ id: 'my-plugin', badge: '3' })
```

## Source

- [`packages/kit/src/node/host-docks.ts`](https://github.com/vitejs/devtools/blob/main/packages/kit/src/node/host-docks.ts) — `DevToolsDockHost.update` throws `DTK0052` when the supplied entry's id isn't in `views`.
