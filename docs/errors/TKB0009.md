# TKB0009: Command Already Registered

> Package: `takubox`

## Message

> Command "`{id}`" is already registered

## Cause

`DevToolsCommandsHost.register()` rejects duplicate command ids.

## Fix

Use a unique id or call `unregister(id)` first.

## Source

`packages/takubox/src/node/host-commands.ts`

## Migrated from

Previously documented as `DTK0024` under `@vitejs/devtools`.
